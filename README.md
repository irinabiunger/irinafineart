# Irina Fine Art Website

This is the starter website for Irina Biunger's artist portfolio.

## Files

- `index.html` — all website text and links
- `style.css` — design
- `script.js` — mobile menu and current year
- `images/` — replace the placeholder images with photos of real paintings

## Replace an image

The easiest method is to upload your image with exactly the same filename:

- `hero.svg`
- `wildlife.svg`
- `seascape.svg`
- `flowers.svg`
- `landscape.svg`
- `artist.svg`

You can also use JPG files. In that case, edit `index.html`, for example:

`images/hero.svg` → `images/hero.jpg`

## Important

In `index.html`, replace:

- `YOUR-EMAIL@example.com`
- the Etsy `href="#"`
- the eBay `href="#"`

Do not upload passwords, private documents or other confidential information.
